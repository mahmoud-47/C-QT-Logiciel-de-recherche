# Projet Developpement Logiciel avec C++ et QTCreator
Gainde Sett est un logiciel qui utilise les tableaux de hachages pour rechercher des structures dans une liste chainée. 
L'idée est de mettre en évidence l'optimisation des algorithmes de recherche en utilisation une structure avancée telle que les tableaux de hachages
